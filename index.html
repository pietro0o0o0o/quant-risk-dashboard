// Features

01 · RISK

VaR & CVaR

Historical simulation Value at Risk and Conditional VaR (Expected Shortfall) at configurable confidence levels.

02 · PERF

Performance metrics

Sharpe, Sortino, Beta, Alpha, rolling correlation vs benchmark. Annualized figures with risk-free rate input.

03 · DIST

Distribution analysis

Return histogram with Jarque-Bera normality test, skewness and excess kurtosis to detect fat tails.

04 · MC

Monte Carlo simulation

Geometric Brownian Motion price path simulation with 5th/95th percentile bands and P(profit) estimate.

05 · DATA

Live market data

Real-time prices via Yahoo Finance. Supports any global equity ticker. Data cached for performance.

06 · VIZ

Interactive charts

Fully interactive Plotly charts — zoom, pan, hover. Drawdown, cumulative returns, rolling correlation.

// Methodology

VaR_α = −Q_α(r_t)

Historical simulation VaR — α-quantile of the empirical return distribution.

CVaR_α = −E[r_t | r_t ≤ VaR_α]

Expected Shortfall — mean loss conditional on exceeding VaR.

Sharpe = (R_p − R_f) / σ_p

Annualized risk-adjusted return relative to configurable risk-free rate.

S_{t+Δt} = S_t · exp[(μ − σ²/2)Δt + σ√Δt · Z], Z ~ N(0,1)

Geometric Brownian Motion — Monte Carlo price path simulation.
